\section{Abstract}
Magnetoencephalography (MEG) data is classified into one of four states (rest, math, motor, memory) using a one-dimensional convolutional neural network (1D-CNN).

\section{Introduction}
Magnetoencephalography (MEG) is a neuroimaging technique that maps brain activity.
Here, sensors are placed on the scalp that measure magnetic fields generated by neurons.
MEG-data is used in neuroscience research and allows for the detection of mental disorders as well as epilepsy.
We will choose a deep learning model that is capable of using MEG-data to classify whether a subject is in one of the following four states: rest, math, memory, or motor.
Two types of classifications are performed in brain decoding: intra-subject classification and cross-subject classification.
Intra-subject classification uses deep learning to train and test a model on the same subject.
For cross-subject classification, a model is first trained on a group of subjects, after which it is tested on a new unseen group of subjects.
We need to perform both types of classification and subsequently compare their respective accuracy.
We will also explain our choices of hyper-parameters regarding the model architecture, and investigate how they affect the model's performance.

\section{Related Work}
Recent advancements in brain decoding have increasingly favored deep learning approaches over traditional machine learning methods that rely on hand-crafted features.
Deep neural networks, particularly Convolutional Neural Networks (CNNs) and Recurrent Neural Networks (RNNs), are capable of directly extracting complex spatiotemporal features from raw Magnetoencephalography (MEG) and Electroencephalography (EEG) signals.
For instance, compact architectures inspired by EEG studies have been successfully adapted to MEG data to decode various mental states, demonstrating superior feature representation abilities \cite{Raza2023}.

A significant challenge in neural decoding is the distinction between intra-subject and cross-subject classification.
Intra-subject models generally achieve high accuracy as they are trained and tested on the specific functional mappings of a single individual.
However, cross-subject classification suffers from severe performance degradation due to inter-subject variability in brain anatomy, cognitive strategies, and signal-to-noise ratios.
Studies have shown that deep learning models trained on specific subjects often overfit to subject-specific features and struggle to generalize to unseen subjects without the use of extensive alignment strategies or massive training sets \cite{Frontiers2020}.
To bridge this gap, various architectures, such as hybrid gated recurrent networks, have been proposed specifically to tackle inter-subject MEG decoding by learning more robust, subject-invariant representations \cite{Li2021}.

Furthermore, the classification of distinct cognitive and motor tasks remains a core focus in Brain-Computer Interface (BCI) research.
Recent work utilizing MEG has demonstrated the feasibility of decoding mental imagery tasks such as cognitive processes (e.g., mathematical subtraction, working memory) and motor execution, on a single-trial basis \cite{Raza2023}.
These studies emphasize that while cognitive and motor tasks elicit distinct neural signatures, the varying representational structures require models that can effectively map both the localized sensor activations (such as high-intensity peaks during math tasks) and distributed patterns associated with resting states.


\section{Approach}
The data is MEG signals that have been gathered from test subjects either in a state of rest or preforming a math and story task, a working memory task or a motor task.
There is one test and train set for intra-subject classification, and one train set and three test sets for cross-subject classification. We will treat the two types of classifications as separate experiments, but use the same model for both.
Each set has 35624 recorded timesteps for 248 magnetometer sensors placed on the human scalp.
During preprocessing, the data is downsampled with a factor of 10 using the average of the signal. We apply z-score normalization using the StandardScaler from scikit-learn: one fit for the intra-subject sets and one for the cross-subject sets.

The model we used to classify the signals is a one-dimensional convolutional neural network (1D-CNN). Compared to other Neural Networks, 1D-CNN has significantly reduced computational cost with only a slight reduction in classification accuracy \cite{app15073593}.
We use the Adam optimizer and Cross Entropy Loss from the Torch library. The neural network, also implemented with Torch, consists of two convolutional blocks that each is followed by ReLU and a max pooling operation, to reduce the temporal resolution.
The network then applies global average pooling and a fully connected classifier with a ReLU activation. To reduce overfitting, the network then applies dropout, and finishes with a fully connected layer that results in logits for the four target classes.
The computational flow of the network is
\[
\begin{aligned}
\text{Input}
&\rightarrow \mathrm{Conv1D}(248\rightarrow32,kernel\_size=5,padding=2) \\
&\rightarrow \mathrm{ReLU}
\rightarrow \mathrm{MaxPool1D}(4) \\
&\rightarrow \mathrm{Conv1D}(32\rightarrow64,kernel\_size=3,padding=1) \\
& \rightarrow \mathrm{ReLU}
\rightarrow \mathrm{MaxPool1D}(4) \\
&\rightarrow \mathrm{AdaptiveAvgPool1D}(1)
\rightarrow \mathrm{FC}(64\rightarrow32) \\
&\rightarrow \mathrm{ReLU}
\rightarrow \mathrm{Dropout}(p)
\rightarrow \mathrm{FC}(32\rightarrow4).
\end{aligned}
\]

During training, we apply hyperparameter tuning using grid search on dropout rate (for CNN), learning rate (for the optimizer), and batch size, which regards the batches in which the data is loaded.
The hyperparameter configuration that has the highest accuracy during training is used for testing. We train and test the model for intra and cross-subject classification separately.
For evaluating the performance of the model we use the measures precision, recall and accuracy.

\subsection{Sensor Saliency}

Fig.~\ref{fig:saliency} shows sensor saliency for the cross model computed via input gradients. The distribution of gradient magnitudes is smooth and gradual across all four classes, indicating that no single sensor dominates the classification. The math class shows a notably higher peak gradient magnitude ($\sim\!1.3 \times 10^{-9}$) compared to the other classes ($\sim\!3$--$8 \times 10^{-10}$), which suggests the model relies on a somewhat more localized set of sensors when classifying math, while the remaining classes are handled through a more distributed pattern across the sensor array.

\section{Conclusion}
In this study we built a deep learning model to classify MEG data into one of four possible states: resting, memory, math, and motor.
Two types of classification tasks were performed, these are intra-subject classification and cross-subject classification.
We were given one training and test dataset for intra-subject classification.
For cross-subject classification we were provided with one training set and three test sets.
Preprocessing consisted of downsampling and z-score normalization.
For classification we used a 1D-CNN along with the Adam optimizer and Cross Entropy Loss.
The neural network consisted of two convolutional blocks, each followed by a ReLU and max pooling operation.
This is followed by global average pooling and fully connected layer that uses ReLU.
Dropout is subsequently applied after which there is a final fully connected layer.
After training, the model achieved a 100\% on the intra-subject test set.
Cross-subject test sets 1 and 3 both achieved 87.5 \% accuracy.
However, cross-subject test set 2 was not able to exceed the chance baseline of 25\%.
We estimate that this issue stems from overfitting.
Overall, we conclude that 1D-CNNs are well suited for intra-subject classification of MEG data.
The results also suggest that cross-subject is a bit more difficult, though not impossible.
